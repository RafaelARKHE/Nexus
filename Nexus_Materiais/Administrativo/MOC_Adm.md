# MOC — Administrativo

> Índice de documentos institucionais do CFO T16.
> Categorias criadas sob demanda — novas pastas surgem conforme novos tipos de documento chegam.
> Antes de criar uma nova categoria, verificar se alguma existente já se encaixa.

---

## Como usar este MOC

Ao receber um documento administrativo novo:
1. Ler este MOC para ver as categorias existentes
2. Decidir se o documento se encaixa em alguma categoria
3. Se sim → mover para a pasta correspondente e registrar abaixo
4. Se não → criar nova pasta com nome descritivo, mover, registrar abaixo
